# Learnings

## Code Quality
- Don't rebuild derived maps in hot paths — cache on `id(obj)` at module level (per-call `inspect.getsource` = 50×+ regression). *(2026-04-23 10:00 ET)*
- **No `load_block`/`matmul_block`/`allocate_buffers` gadgets** — tile loops IR-positioned; bare `nl.ndarray`+`nisa.dma_copy`/`nc_matmul` at op fire depth. Transpose stays its own op. *(2026-04-26 11:00 ET)*
- **Loud failures only**: no silent raises, no `try/except` to adapt around malformed IR — reject bad input. A transform that no-ops on bad input yet returns "success" is the same violation. *(2026-06-01 22:00 ET)*
- **Single return per function** — user rejects branching returns in codegen helpers; build a result var, return at bottom. *(2026-05-15 22:00 ET)*

## Architecture
- **`nkigym/__init__.py` lazy `__getattr__`**: eager re-exports drag heavy deps (`claude_agent_sdk`) into gym workers — keep `__init__` empty of compile/synthesis loads. *(2026-05-05 18:00 ET)*
- **Editable pip install pins source path** across shared-venv worktrees: subprocess tests (`sys.executable`) pick up the installed path not cwd → set `PYTHONPATH=<worktree>/nkigym/src`. *(2026-05-07 13:00 ET)*

- **Neuron allocator** Chaitin-Briggs+zone-aware: `address=` annotations skip `AddressRotation::run` → ~48pp MFU loss. `KernelJob.neuronx_cc_args` defaults `()` — pass scheduler-off flags (PSUM-OOM) with `remote_profile`. *(2026-04-26 04:00 ET)*
- **Material-blocking**: reading PSUM w/ open material-blocking → `NCC_ISCH714`. Memset PSUM only when `PSUM_DTYPE=fp32`; transpose bf16 PSUM must NOT memset → `NCC_IBIR441`. HW mandates fp32 for `tensor_scalar.operand0/1`, `activation.scale`. *(2026-04-23 10:00 ET)*
- **Path B (own IR) > Path A (fork neuronx-cc)**: Path A ceiling = compiler IR (no online fusion/rotation/SW-pipeline/PSUM-bank addrs). **Online fusion is NO compiler's transform** (TVM `CommReducer` fixed-associative) — *ours* regardless. Adopting TVM rejected (TIR→NKI backend unbuilt; agent-legibility). **Decision: faithfully PORT TVM's schedule transforms into the custom IR** (BUILT apache/tvm main to confirm). *(2026-06-03 02:00 ET)*
- **arith substrate SHIPPED (Spec 1 of 4 of the TVM port; `nkigym/ir/arith/`)**: faithful TVM `src/arith/` port — `expr`, `rewrite_simplify`, `analyzer`, `iter_map` (`detect_iter_map`/`iter_map_simplify`), `int_set`. **Oracle-verified vs a BUILT TVM**; TVM dep **test-only** (`importorskip`), shipped `nkigym/` imports zero tvm. **Split/Reorder/Fuse were ALREADY arith-thin** — tree-surgery delegating affine work to `normalize_block`; got oracle GUARDS, not rewrites (consumer story in sub-bullet). `can_prove_equal` can't fold `(f//c)*c+f%c→f` — use `detect_iter_map`. *(2026-06-03 12:00 ET)*
  - **Arith-thin split/fuse/reorder SHIPPED — only Split cover check migrated** (`_covers_exactly`: `const_int_bound`, `hi+1==extent`; KEEP exact `Π==extent`, NOT TVM's predicated `Π≥extent`). **NONE of the 3 NEEDS arith computationally**: Split's `const_int_bound` is *provably ≡ `prod(factors)==extent`* — kept as TVM-faithfulness mirror; Fuse uses `Expr` annotation only; Reorder zero. **compute_at `_domain_solve` is the FIRST genuine arith need** (Spec 2). Part A DROPPED. `normalize_block` SHARED by Split/Fuse AND `_code_motion`. *(2026-06-03 16:00 ET)*
- **Repo scope (post 2026-05-13 restart)**: `NKIOps` + `KernelIR` build + `render` to NKI source. `KernelIR` (`ir/ir.py`) wraps `DimensionAnalysis` + `KernelTree`; `.dump(cache_dir)` emits markdown + tree PNG + `kernel.py`. Single example `examples/matmul_lhsT_rhs.py`. *(2026-06-01 22:00 ET)*
- **NO trip-1 ForNodes anywhere**: a dim's iteration = factorization `(trip₀…trip_{k-1}, tensorize_size)`, product = dim extent. ForNode exists ONLY when trip > 1; innermost `tensorize_size` = ISA-leaf width. *(2026-06-01 02:00 ET)*
- **BlockNode IR** (`tree.py`, nx.DiGraph): payload union `BlockNode|ForNode|ISANode`; payloads own `.label()`. A `BlockNode` (TVM `SBlock`) owns `iter_vars` (axis+`dom=(lo,hi)`+role), `iter_values` (affine loop_vars→iter_var), `reads`/`writes`, `alloc_buffers`. *(2026-05-29 09:00 ET)*
- **`BlockNode.axis_map`** (concrete→abstract): tensorize/Split translate `d2`→op-axis `N` for `OPERAND_AXES`/`MIN_TILE_SIZE`. *(2026-06-01 22:00 ET)*
- **SSA ops backend**: `f_nkigym` is SSA — `out = NKIOp()(src=…)`, allocate-and-return; `NKIAlloc`/`dst=`/authored memset GONE. Memset synthesized in `canonical_build` from `RMW_OPERANDS` (matmul only). `NKITensorCopy` KEPT explicit (multi-output breaks SSA). *(2026-06-01 22:00 ET)*
- **Two-line dtype rule** (`Buffer.physical_dtype`): (1) PSUM accumulation alloc ALWAYS fp32 — the ONLY hardcode; (2) every other buffer = FIRST HBM input's dtype. *(2026-06-01 02:00 ET)*
- **Three redundant-but-load-bearing layers** per BlockNode iteration: `iter_var.dom` (full extent), ForNode trips, `BufferRegion (lo,width)`. Invariant `Π(trip)×width == dom`; redundancy serves compute_at region-regen. *(2026-05-29 09:00 ET)*
- **LCA buffer placement, re-derived after EVERY transform** (`place_buffers`): each `Buffer` on the LCA BlockNode of all blocks touching it — the PSUM-hoist lever (compute_at co-locating memset/matmul/tensor_copy under shared M,N drops the LCA). `shared_hbm` ALWAYS pinned at root (else re-alloc per iter → NaN). *(2026-06-02 10:00 ET)*
- **Location enum: `{shared_hbm, sbuf, psum}`** (not `hbm`); the literal IS the variable prefix. Renderer keys operand dimensionality off `location` — HBM 2D, SBUF/PSUM 3D. *(2026-05-15 23:00 ET)*
- **Producer-consumer deps = sidecar DAG** (`ir/dependency.py`, networkx; ISA-leaf/ForNode nids). Tree stays block-structural — don't reparent loops to encode RAW. **ONE notion: same buffer + overlapping region + RAW/WAW/WAR**. "CARRY" KIND dead; legality = leaf positions + spans. *(2026-06-03 02:00 ET)*
  - **Ill-defined-edge law**: `a→b` is meaningless when one nests inside the other → NO node↔enclosing-loop edge. Resolution = **symbolic unroll**: graph pure instruction→instruction; loop-carry lives in the CHECK via span-promotion (carried-reducer endpoint → accumulation-loop span).
  - **THE direction bug — FIXED** (`df43f44`): NEVER rebuild `Dependency` on the MOVED tree — tautological (`_build` orients edges by exec order → producer sunk past consumer re-derives RAW as forward WAR → `first_backward_edge` sees nothing → NaN). **Fix**: freeze directions from the ORIGINAL `ir.dependency`, eval positions on the moved tree via a **pure topological query** `first_backward_edge_for_insertion` (no deepcopy/`_move`); realizability split to pure `_check_move_realizable`. **arith = WRONG layer** for ordering (affine only; arith = edge-creation region-overlap + the move's `_domain_solve`).
  - **Carry axis = role-based**: `producer→outermost-loop-of-axis` edge only when the enclosing loop is non-PARALLEL (ACC/SEQ) AND axis absent from output chunk. Irrelevant PARALLEL wrap → NO edge. SEQ ⊇ ACC.
- **Codegen direction: ir → codegen, never reverse.** Renderer takes IR primitives (tree/block nodes, `BufferRegion`s), not `KernelIR` — one import direction kills circular deps. Walk emits every loop, shape from `Buffer`. *(2026-05-15 19:00 ET)*
- **Renderer mechanics** (`codegen/body.py`): `_emit_block`/`_emit_subtree` recurse from `tree.root`. **loop_var naming = model B**: dense `i_d{dim}_{N}`, N = ordinal among that dim's ForNodes outer→inner. *(2026-06-01 04:00 ET)*
- **Transforms shipped: Split + Fuse + Reorder + ComputeAt/ReverseComputeAt** (`nkigym/transforms/`). `Transform` base: `analyze(ir)→list[TransformOption]`, `apply(ir,option)→KernelIR` (deep-copy + mutate, re-checks legality → loud `TransformLegalityError`). *(2026-06-02 10:00 ET)*
- **Split must respect `MIN_TILE_SIZE`** (matmul K/M/N all 128): tensorize-split legal iff innermost factor `>= MIN_TILE_SIZE[axis]`; gated in `analyze` + `_check_legality`. M/K@128 unsplittable; N(512)→128 only. *(2026-06-01 22:00 ET)*
- **compute_at SHIPPED** (`dev_1`): shared `_move(...,is_reverse)`, two faces differ ONLY in dep-check direction. **TVM region-regen, NOT trip-collapse**: `SolveBlockVarDomain` inverts access vs required region; `MakeNewLoop` regenerates residual loops. PSUM hoist *composed* (sink memset + lift tensor_copy → LCA descends). **Region-regen invariants**: gather enclosing-ancestor loops (else offset→0→NaN); residual dense-naming continues the dim ordinal across the enclosing chain; `compact_shapes` anchor = loops enclosing **all** touchers, offsets-identical. *(2026-06-02 10:00 ET)*
- **Sibling order is dataflow**: `nx.DiGraph.add_edge` appends to the successor list — naive remove-then-add moves the chain to the END (load rendered AFTER matmul). Splice at the original child index. *(2026-05-17 14:00 ET)*
- **Memset = sibling block, NOT nested `matmul.init`** (`BlockNode.init` dropped; folding gave duplicate loops + INVERTED edge). init/compute/drain at THREE loop levels = TVM's *post-`decompose_reduction`* state. Cost: a free memset CAN sink into K → needs the `memset→K-loop` RAW edge to forbid. *(2026-06-03 02:00 ET)*
- **Op axis names follow `OPERAND_AXES`, not matmul names**: `NKILoad.OPERAND_AXES` is `{"P","F"}`, so a load's iter_var axes are `{"P","F"}` even when concrete dim is `d1`/M. Tests index load regions with `"F"` not `"M"`. *(2026-05-17 14:00 ET)*
- **Loop-role placement (OP-LOCAL)**: `ForNode` role-neutral; role on `BlockNode.iter_var.role` (=`AXIS_ROLES[axis]`, default PARALLEL); same dim differs across ops. Split/Fuse role-blind; Reorder/compute_at consult it. *(2026-05-29 09:00 ET)*
- **Reorder transform**: payload-swap adjacent parent-child ForNode pair (self-inverse). **Legality**: perfect-nest; no descendant ISA leaf SEQUENTIAL on either swap_dim. *(2026-05-26 22:00 ET)*
- **MFU SOTA vs nkipy baselines** (`@baremetal_jit`→HLO→`neuronx-cc -O2`): matmul_lhsT_rhs 90.92% (>86.65%); matmul_lhs_rhs 84%; rmsnorm+matmul ~79-80%; causal attn 13.20%. *(2026-04-27 12:00 ET)*
- **Profiler `throttle_avg_util_limit` NOT HW ceiling** (activity-weighted); real gap = `tensor_engine_active_time_percent`. **Roofline efficiency = `100*MFU/mfu_max_achievable_estimated_percent`** — low+low = tuning headroom, low+high = mem-bound. *(2026-05-05 05:00 ET)*
- **dma_transpose** HBM→SBUF AND SBUF→SBUF, enforces matching dtypes (`dma_copy` tolerates mismatch). `nc_transpose(dst=psum)` IS `nc_matmul(moving=identity, is_transpose=True)`, kwarg `data=` NOT `src=`. `nl.divide` rejected → `reciprocal+multiply`. ## Verification
- Multi-D PSUM trips sim reshape. HW rejects 4D affine-select dst APs (`NCC_INLA001`). `NCC_ISCH714` fires on aliased `address=` RAW inorder. `nisa.dma_copy` has NO transpose — use `nisa.dma_transpose(dst,src,axes,dge_mode)`. *(2026-04-23 10:00 ET)*
- **Byte-exact transform verification**: gate = `render(apply(before_IR)) == <hand kernel>`, NOT "roughly the same" (user: "same means same"). Oracle (`_ladder_compare.py`) is **AST-canonical** (hoists decls, positional→keyword, sorts kwargs, distributes affine). Topology/sim-only tests hide region bugs — render+sim caught all 5 move-engine bugs. *(2026-06-02 10:00 ET)*
- **Numeric sim is a WEAKER oracle than dependency-order** — legality = dep-order preservation, NOT numeric luck. memset-after-matmul / drain-sunk-into-K sim "correct" by last-write-wins coincidence yet violate producer/consumer order + break on HW. A check rejecting them is RIGHT even when sim passes. *(2026-06-03 02:00 ET)*
- **Behavior-preserving refactor of a legality check → GOLDEN-VERDICT gate, not the suite**: swapping deepcopy+`_move` for a pure topological query silently over-permitted 180 then 52 candidates (`illegal→ok`) while the full suite + 16 sim kernels stayed GREEN. Capture every candidate's old verdict first, require exact match — caught 3 subtle position-math bugs the suite missed. *(2026-06-04 16:00 ET)*
- **FP32 validation contract**: CPU-sim runs fp32 (inputs cast, SBUF/PSUM→`nl.float32`, fp32 golden); HW keeps original. End-to-end `atol=rtol=5e-3` (`assert_allclose` raises+halts, so N PASS + exit 0 = all passed); unit stricter — elementwise `1e-6`, matmul K≤4096 `1e-5`, reduce K≥16384 `1e-4`. *(2026-05-06 17:00 ET)*
