# Learnings

## Code Quality
- Don't rebuild derived maps in hot paths. Cache on `id(obj)` at module level. Per-call `inspect.getsource` was 50×+ regression. *(2026-04-23 10:00 ET)*
- **No `load_block`/`matmul_block`/`allocate_buffers` gadgets** — tile loops are IR-positioned. Bare `nl.ndarray`+`nisa.dma_copy`/`nc_matmul` at op fire depth. Transpose stays its own op (real ISA distinction). *(2026-04-26 11:00 ET)*

## Architecture
- `nkigym` core must NOT import from `autotune` — one-way boundary. Keep local helpers even when crossing it would dedupe. *(2026-04-23 10:00 ET)*
- **`nkigym/__init__.py` lazy `__getattr__`**: eager re-exports of `nkigym_compile` drag `claude_agent_sdk` into gym workers (they only need `nkigym.ops`). `__init__` must not eagerly load `compile`/`synthesis`. *(2026-05-05 18:00 ET)*
- **KernelIR = flat dataclass**. Per-op resolved data (`op_inputs`, `op_outputs`, `op_kwargs`, `op_axis_map`, `op_tile_sizes`, `op_blocking_dims`) keyed by `NKIOp` id. `logical_tensors` = math-level pre-DMA; `physical_buffers` = DMA aliases (`<param>_sbuf`). *(2026-04-23 20:00 ET)*
- **`nl.ndarray` shape is 3D**: `(P_tile, num_p_tiles, num_f_tiles*f_tile)`. No `nl.par_dim(128)` — bare `128`. Prefer `nisa.dma_copy` over `nl.load`. Always explicit ranges (`0:128`, never `:`). Parenthesize compound `f_slot` when multiplying. *(2026-04-26 14:00 ET)*
- **Neuron allocator**: Chaitin-Briggs+zone-aware. `address=` annotations = binary poison → skips `AddressRotation::run` → ~48pp MFU loss. Scheduler-off flags prevent PSUM-OOM; `KernelJob.neuronx_cc_args` defaults `()`. *(2026-04-26 04:00 ET)*
- NKI HW mandates fp32 for `tensor_scalar.operand0/1`, `activation.scale`, `activation_reduce.scale`. `_promote_float32_consumers` handles. *(2026-04-23 10:00 ET)*
- **Material-blocking**: reading PSUM w/ open material-blocking loop → `NCC_ISCH714`. Memset PSUM only when `PSUM_DTYPE=fp32`. Transpose bf16 PSUM must NOT memset — `NCC_IBIR441`. *(2026-04-23 10:00 ET)*
- **Online fusion Alg 2**: `(op_X, op_A)` — op_X∈{SEQ,ACC}; op_A strictly ACC. Role lattice `PAR⊂SEQ⊂SUM`. Detector `{SEQ}→PAR*→SUM`. Pre-pass `HoistDivideThroughContraction` hoists softmax `÷l` past MM2. *(2026-05-01 23:00 ET)*
- **HLO algsimp folds flash-identity constants** → flash w/ `m=-inf,l=0,O=0` Python consts collapses to vanilla. Workaround: promote state to kernel inputs (Parameter HLO ops). *(2026-04-29 01:00 ET)*
- **`dim_role` OP-LOCAL**: each op carries `dim_role={...}`; same dim can have different roles across ops. *(2026-04-25 00:00 ET)*
- **Path B (own IR stack) > Path A (fork neuronx-cc)**. Path A ceiling = compiler's internal IR (can't express online fusion, rotation, SW pipeline, PSUM-bank addrs); Path B = hand-tuned by construction. Source-level emit avoids fork maintenance. *(2026-05-04 05:00 ET)*
- **Bottom-IR target**: hand-stripped `attention_cte.py` flash slice (bs=1, seq=16384, d=64, fp16, causal). Single `@nki.jit def`. 1.947ms / 44.88% MFU. Verbatim repro: `canonicalize(render(IR)) == canonicalize(manual)`. *(2026-05-04 05:00 ET)*
- **`trace` (semantic) + `canonicalize` (spelling) = two passes**. Trace to FIXPOINT (inline + dataclass flatten + constant fold + DCE). Canonicalize: role-based rename, import sort, kwarg order. Dataclass-stripping only touches classes in the module under transform. *(2026-05-05 00:00 ET)*
- **Canonical role-based naming**: loop vars `i_<N>`, SBUF `sbuf_buffer_<N>`, PSUM `psum_buffer_<N>`, HBM `hbm_buffer_<N>`, library instances `object_<N>`, other locals `local_<N>`. Deterministic. Params + allowlist imports (`nki`, `nkilib.core.utils.*`, `numpy`, `dataclasses`, `typing`) preserved. *(2026-05-05 00:00 ET)*
- **MFU SOTA**: matmul_lhsT_rhs 90.92% vs nkipy 86.65%; matmul_lhs_rhs 84.06% (dma_transpose > nc_transpose PSUM-staged); rmsnorm+matmul 79.09%/80.57%. Rotation NOT primary lever. *(2026-04-27 12:00 ET)*
- **Profiler `throttle_avg_util_limit` NOT HW ceiling** — activity-weighted; real gap = `tensor_engine_active_time_percent`. *(2026-04-24 10:00 ET)*
- **Architecture boundary**: `remote_profile` IR-agnostic (takes `KernelJob`=rendered source); `autotune` must NOT import nkigym codegen. `write_kernel_sources` before `profiler.profile`. *(2026-05-05 05:00 ET)*
- **dma_transpose** works HBM→SBUF AND SBUF→SBUF; enforces matching dtypes (`dma_copy` tolerates mismatch). `nc_transpose(dst=psum)` IS `nc_matmul(moving=identity, is_transpose=True)`; kwarg is `data=` NOT `src=`. `nl.divide` in tensor-valued `tensor_scalar` rejected — use `reciprocal+multiply`. Op tile limits: Load/Store/DMATranspose P=128; nc_transpose P=128,F=128; nc_matmul K=128,M=128,N=512. *(2026-04-25 03:00 ET)*
- **ISA**: `nisa.activation_reduce` free-axis cap = **16384 per partition** (existing `VE_FREE_MAX=512` WRONG — that's `bn_stats`/`nc_matmul.moving`). Hardwired `reduce_cmd=reset_reduce`. Production patterns: **P1** RMSNorm-family = one call over full F + `activation(rsqrt,scale,bias)` post-op (2 ops); **P2** online softmax = tile F, each call writes distinct `reduce_res` slot, close via ONE `tensor_reduce(axis=1)`. Production NEVER SBUF-merges `tensor_tensor` across F-tiles. `activation_reduce` requires `dst.shape == data.shape` → zero-stride AP `.ap(pattern=[[1,P],[0,F]])` keeps scratch at `(P,1)`. `tensor_scalar.operand0` broadcasts along FREE. *(2026-05-05 20:00 ET)*
- **nkilib RMSNorm+quant**: `activation_reduce(square,add)` → `activation(rsqrt,scale=1/H,bias=eps)` → `scalar_tensor_tensor(mul,mul)` fuses `x·inv_rms·γ`. *(2026-05-05 00:00 ET)*
- **nkipy baselines**: `@baremetal_jit`→HLO→`neuronx-cc --internal-tensorizer-opt-level=2`. 2048³ lhsT@rhs=86.65%; rmsnorm+matmul=80.57%; causal attn 2048×128=13.20%. `lower_to_nki(trace, save_artifacts=True)` dumps NKI. *(2026-04-25 04:00 ET)*
- **Attention MFU** (trn2, opt-level=2): compiler flash/vanilla non-monotonic, neither >~18%. Hand `nkilib.attention_cte` D=128 bf16 30/46% @ 2k/16k; D=64 fp16 flash seq=16k → **44.88% MFU**. **MAC = 4·SEQ²·D** (MM1+MM2). *(2026-05-04 05:00 ET)*
- **MAC counting convention**: `compute_mac_count` returns `K·M·N` per matmul (1 MAC = 1 mul+1 add = 2 FLOPS). MFU divides by MACs, NOT FLOPS. *(2026-04-29 01:00 ET)*
- **`@nki.jit` nesting**: nested calls fail (inner `shared_hbm` rejected). Wrapper pattern: outer `@nki.jit`, call `kernel.func` (unwrapped). `functools.partial` rejected. **Raw call-path ~20 sec/iter** (re-traces) — use `BaremetalExecutor.benchmark(mode="device")`. *(2026-05-04 05:00 ET)*
- **Compiler-emitted NKI NOT re-shippable**: `remote_numpy_baseline` emits `@trace` not `@nki.jit`. Read as reference, hand-author fresh with `INPUT_SPECS`. *(2026-05-01 23:00 ET)*
- **Synthesis**: `compile_numpy_to_nkigym(f_numpy, input_specs)` single-call. `ClaudeSDKClient` conversational loop — agent emits ```python``` fence/turn, validator feeds result back; `max_attempts` gate. Orchestrator steps in prompt constants (NOT SKILL.md). No worked examples (cheating). *(2026-05-05 00:00 ET)*
- **`@nkigym_kernel` + `_RoleArray`**: runtime role tracking (`param|sbuf|stored|psum`) in `NKIOp.__call__` raises `TypeError` at offending line for raw-param→compute. Decorator tags inputs `param` on entry, asserts return `stored`. `compute_golden` takes plain numpy. *(2026-05-05 02:00 ET)*
- **NKIOp classes mirror NKI ISA 1:1**: no invented kwargs or ops — every DSL op's kwarg set must match its `nisa.*` primitive's signature. `NKIActivationReduce` accepts only `{op, reduce_op}` (matches `nisa.activation_reduce(dst, op, data, reduce_op, reduce_res, bias, scale, name)` minus the generated operands). Fused closures spell as multiple DSL calls: RMSNorm's `rsqrt(sum·scale+bias)` = `NKIActivationReduce(op=square, reduce_op=add)` + `NKIActivation(op=rsqrt, scale, bias)`. *(2026-05-05 23:00 ET)*
- **Mode A vs Mode B**: A = mechanical per-op nest + FuseLoops/ReorderLoops (`nkigym/codegen`). B = agent synthesis via `compile_numpy_to_nkigym` (fresh `@nki.jit`). A ceiling = multi-pass-reducer gap. Use A for single-reducer workloads (matmul 84–90%, rmsnorm+matmul 79–80%); B for attention. *(2026-05-05 23:00 ET)*
- **Multi-buffer + SW pipelining = TWO orthogonal atoms**: `ShrinkBuffer(tensor, dim, N)` (per-dim via LoopNode decoration) + `SoftwarePipeline(fused_loop_path, depth D)`. Constraint `N >= D+1` (slack legal). SW pipeline = prologue/body/epilogue source transform with iteration offsets on BodyLeaf (NOT scheduler, NOT unroll). Shrink legality: producer/consumer share `dim_id` ancestor AND producer→consumer distance `< N` — RAW hazard is CORRECTNESS, caught at `is_legal`. *(2026-05-06 10:00 ET)*
- **Topological adjacency for fusion** (supersedes literal): sibling list = topo-sort of DAG. `FuseLoops(path, boundary=(i,j), dim_id)`: legal iff three-field rule + every intervening `k` commutes with BOTH `i` AND `j` (two-sided — `i→k` edge possible). j=i+1 byte-identical to old. Apply = consumer absorbs producer; fused lands at `j`'s slot; body = `[*producer.children, *consumer.children]`; intervening siblings pushed left preserving order. *(2026-05-06 13:00 ET)*
- **Dep DAG from OpGraph**: tensor names in `ParsedOp.output_names` / `operand_names` are canonical. Materialize `producer` + `consumers` dicts at `parse_and_resolve`; attach as `OpGraph.dep` (new `nkigym/codegen/dep_graph.py`). Reads = `OPERAND_AXES.keys()`, writes = `OUTPUT_AXES.keys()`. `BodyLeaf.op_idx` bridges forest→graph. Minimal sibling perturbation (preserve original order) — `hash_forest` requires deterministic apply. **Circular import gotcha**: `graph→dep_graph→loop_forest→graph`; project bans `TYPE_CHECKING` guards (see code_style), so the pragmatic fix is lazy function-level import of `BodyLeaf` in subtree helpers (once at top of public entrypoint, not per-recursion — Python `sys.modules` cache makes it cheap). *(2026-05-06 16:00 ET)*
- **Eager codegen overview**: per-op independent loop nests; intermediates full-extent 3D SBUF. AST-parse > tracing (synthesis prompt forbids `if`/`for`/helpers). Compute-skip = post-parse annotation (`per_op_attrs`); renderer wraps annotated body in 3-branch `skip_all/compute_only/mask_and_compute`. *(2026-05-05 03:00 ET)*

## Verification
- Multi-D PSUM trips sim reshape. HW compiler rejects 4D affine-select dst APs (`NCC_INLA001`). `compile_bir_to_neff` catches `RuntimeError` into `neuronx_cc_error`; `_run_compiler` MUST append stderr. `NCC_ISCH714` fires on aliased `address=` reader-after-writer `inord`; strip `address=` to diagnose. `nisa.dma_copy` has NO transpose — use `nisa.dma_transpose(dst, src, axes, dge_mode)`. *(2026-04-23 10:00 ET)*
- **Sim-path fp32 contract**: CPU-sim runs fp32 (inputs cast, SBUF/PSUM rewritten to `nl.float32`, numpy-fp32 golden). HW: original dtypes. **`_rewrite_to_fp32` matches `dtype=nl.<name>` only** — misses positional dtype args (`load_dtype=nl.bfloat16`). Extend to rewrite `nl.bfloat16/float16/float32` wherever they appear. *(2026-05-04 05:00 ET)*
- NKI CPU sim doesn't quantize on dtype-narrowing copies. 2048³ bf16 `worst_margin` floor ~0.324 at 1e-2; 1e-3 unreachable. *(2026-04-23 10:00 ET)*
- **Roofline efficiency = `100 * MFU / mfu_max_achievable_estimated_percent`**. Low MFU + low roofline = tuning headroom; low MFU + high roofline = memory-bound, need higher AI. *(2026-04-26 00:00 ET)*
- **CST round-trip on `attention_cte.py`** (2982→922 canonical): NKI has only structured `for`+straight-line ISA+`nl.ndarray`+occasional `if`-mask → **tree IR sufficient, CFG not needed**. Flash/MoE/online-fusion/causal-mask tree-expressible. *(2026-05-01 23:00 ET)*
- **Profiler-native metrics only**: `profiler_summary["mfu_estimated_percent"]` (instruction-level) + `total_time` replace custom `calculate_mfu` math. Instruction-MFU ≠ full-square MFU for compute-skipping kernels (causal attn: profiler 35% vs full-square 100%); real gap = `tensor_engine_active_time_percent`. `mac_count` informational. `ProfileConfig` fields flattened onto `remote_profile`/`remote_run`. *(2026-05-05 00:00 ET)*
- **FP32 validation contract**: regardless of `input_specs` dtypes, cast both `f_numpy` reference AND candidate `f_nkigym` to fp32 for cross-check. FP32 tolerance: elementwise `atol=rtol=1e-6`, reduce/matmul K≤4096 `1e-5`, large reduce K≥16384 `1e-4`; numpy's `rtol=1e-7` default too tight for any K≥2048 reduction. *(2026-05-05 00:00 ET)*
