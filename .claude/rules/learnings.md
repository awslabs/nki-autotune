# Learnings

## Code Quality
- Don't rebuild derived maps in hot paths — cache on `id(obj)` at module level (per-call `inspect.getsource` = 50×+ slowdown). *(2026-04-23 10:00 ET)*
- **No `load_block`/`matmul_block`/`allocate_buffers` gadgets** — tile loops IR-positioned; bare `nl.ndarray`+`nisa.dma_copy`/`nc_matmul` at op fire depth. **`nl.ndarray` N-D explicit** `(P, num_p_tiles, F_contig)`; NO trivial-dim folding; bare `128` (no `nl.par_dim`); `nisa.dma_copy`>`nl.load`. *(2026-04-26 11:00 ET)*
- **Loud failures only**: no silent raises, no `try/except` to adapt around malformed IR — reject bad input. A transform that no-ops on bad input yet returns "success" is the same violation. **Single return per function** (user rejects branching returns). *(2026-06-01 22:00 ET)*
- **Transform legality = behavior/dep-order + ISA well-formedness ONLY; NEVER resource capacity** (user-locked): a dep-legal kernel that over-subscribes PSUM/SBUF is a VALID transform output — compile/HW profiling prunes it. DO reject malformed IR / dep violations / ISA-impossible ops (sub-128 `nc_matmul`). DO NOT gate "won't fit in PSUM". *(2026-06-06 06:00 ET)*

## Architecture
- **`nkigym/__init__.py` lazy `__getattr__`**: eager re-exports drag heavy deps (`claude_agent_sdk`) into gym workers — keep `__init__` empty of compile/synthesis loads. **Editable pip install pins source path** across shared-venv worktrees: subprocess tests pick up installed path not cwd → `PYTHONPATH=<worktree>/nkigym/src`. *(2026-05-07 13:00 ET)*
- **Neuron allocator** Chaitin-Briggs+zone-aware: `address=` annotations skip `AddressRotation::run` → ~48pp MFU loss. *(2026-04-26 04:00 ET)*
- **Material-blocking**: reading PSUM w/ open material-blocking → `NCC_ISCH714`. Memset PSUM only when `PSUM_DTYPE=fp32`; transpose bf16 PSUM must NOT memset → `NCC_IBIR441`. HW mandates fp32 for `tensor_scalar.operand0/1`, `activation.scale`. *(2026-04-23 10:00 ET)*
- **Path B (own IR) > Path A (fork neuronx-cc)**: Path A ceiling = compiler IR (no online fusion/rotation/SW-pipeline/PSUM-bank addrs). **Decision: faithfully PORT TVM's schedule transforms into the IR.** Scope: nkigym ≈ pure-Python port of **exactly TVM's TensorIR schedule layer + `arith`** (NOT Relax/TE/topi) — single target; win is online fusion + agent-legibility, not multi-target. *(2026-06-03 02:00 ET)*
- **arith substrate SHIPPED (Spec 1/4 of TVM port; `nkigym/ir/arith/`)**: faithful TVM `src/arith/` port. TVM dep **test-only** (`importorskip`), shipped `nkigym/` imports zero tvm. `can_prove_equal` can't fold `(f//c)*c+f%c→f` — use `detect_iter_map`. *(2026-06-03 12:00 ET)*
  - **Test-slimming = dedup/parametrize, PRESERVE EVERY ASSERTION** (user-locked): `@parametrize` near-identical tests, hoist helpers to `_fixtures.py`.
- **Repo scope**: `NKIOps` + `KernelIR` (`ir/ir.py`, wraps `DimensionAnalysis`+`KernelTree`) build + `render` to NKI; `.dump(cache_dir)` emits markdown + tree PNG + `kernel.py`. *(2026-06-01 22:00 ET)*
- **NO trip-1 ForNodes anywhere**: a dim's iteration = factorization `(trip₀…trip_{k-1}, tensorize_size)`, product = extent; ForNode exists ONLY when trip > 1; innermost `tensorize_size` = ISA-leaf width. **Three redundant-but-load-bearing layers**: `iter_var.dom` (full extent), ForNode trips, `BufferRegion (lo,width)`; invariant `Π(trip)×width == dom`. *(2026-06-01 02:00 ET)*
- **BlockNode IR** (`tree.py`, nx.DiGraph): payload union `BlockNode|ForNode|ISANode`. `BlockNode` (TVM `SBlock`) owns `iter_vars` (axis+`dom`+role), `iter_values` (affine loop_vars→iter_var), `reads`/`writes`, `alloc_buffers`. **Blocks mechanically emitted** (canonical_build = one-per-op; transforms = fixed recipe) — NOT a per-instruction "does it fit" decision; a block carries ONE role per axis, so a differing role-assignment forces its own block. `axis_map` = concrete→abstract (Split translates `d2`→op-axis `N`). *(2026-06-07 20:00 ET)*
- **SSA ops backend**: `f_nkigym` is SSA — `out = NKIOp()(src=…)`, allocate-and-return; `NKIAlloc`/`dst=`/authored memset GONE. Memset synthesized in `canonical_build` from `RMW_OPERANDS`. **Dtype** (`Buffer.physical_dtype`): PSUM accum alloc ALWAYS fp32 (ONLY hardcode); else = FIRST HBM input's dtype. *(2026-06-01 22:00 ET)*
- **LCA buffer placement, re-derived after EVERY transform** (`place_buffers`): each `Buffer` on the LCA BlockNode of all blocks touching it — the PSUM-hoist lever (compute_at co-locating memset/matmul/tensor_copy drops LCA). `shared_hbm` pinned at root (else re-alloc → NaN). **Location enum `{shared_hbm, sbuf, psum}`** (literal IS var prefix); renderer keys operand dims off it (HBM 2D, SBUF/PSUM 3D). *(2026-06-02 10:00 ET)*
- **Producer-consumer deps = sidecar DAG** (`ir/dependency.py`, networkx; ISA-leaf/ForNode nids). Tree stays block-structural — don't reparent loops to encode RAW. **ONE notion: same buffer + overlapping region + RAW/WAW/WAR**; legality = leaf positions + spans. *(2026-06-03 02:00 ET)*
  - **Ill-defined-edge law**: `a→b` meaningless when one nests in the other → NO node↔loop edge. Resolution = **symbolic unroll**; loop-carry via span-promotion. **Carry axis role-based**: `producer→outermost-loop-of-axis` edge only when enclosing loop non-PARALLEL AND axis ∉ output chunk.
  - **Dep directions: freeze from ORIGINAL `ir.dependency`** — NEVER rebuild `Dependency` on the MOVED tree (tautological: no backward edge → NaN); eval positions on moved tree via `first_backward_edge_for_insertion`.
- **Codegen direction: ir → codegen, never reverse.** Renderer takes IR primitives (tree/block nodes, `BufferRegion`s), not `KernelIR` — kills circular deps. **loop_var naming = model B**: dense `i_d{dim}_{N}`, N = ordinal among dim's ForNodes outer→inner. **Sibling order = dataflow**: splice `add_edge` at original child index (else load renders AFTER matmul). *(2026-06-01 04:00 ET)*
- **Transforms shipped: Split + Fuse + Reorder + ComputeAt/ReverseComputeAt** (`nkigym/transforms/`). `Transform` base: `analyze(ir)→list[TransformOption]`, `apply(ir,option)→KernelIR` (deep-copy+mutate, re-checks → loud `TransformLegalityError`). **Split respects `MIN_TILE_SIZE`** (matmul K/M/N all 128) → M/K@128 unsplittable, N(512)→128. **compute_at**: shared `_move(...,is_reverse)`, faces differ ONLY in dep-check direction; **TVM region-regen NOT trip-collapse** (`SolveBlockVarDomain`, `MakeNewLoop`). Can't merge/create blocks (block count invariant) — RFactor makes its own combine/init blocks. *(2026-06-02 10:00 ET)*
- **compute_at TVM-faithfulness (verified `compute_at.cc`/`state.cc`; detail in `tvm_knowledge.md`)**: TVM legality = 4 explicit checks PLUS the load-bearing implicit **`require_stage_pipeline=true`**; the **`region_cover` property** (`state.cc:204`, re-checked every primitive) rejects a producer whose write region fails to cover a consumer's read. **KEY: block-granular deps CAN express init-domination via region-coverage, NOT a loop-node edge** — our 2026-06-02 bug was *absence of region_cover*, not a granularity limit; `_check_no_reduction_axis_covered` is OUR substitute. *(2026-06-09 00:00 ET)*
  - **TVM init = TWO-state model**: default WELDED (`block->init`, no SRef, rides with block, lowers PREDICATED `if reduce==min`); `decompose_reduction(block, loop_rv)` extracts it to a schedulable `SBlockRV`. Placement is **schedule-controlled, not derived**; equal expressiveness to ours. **We're PERMANENTLY post-decompose** → never port init-field/predication/`decompose_reduction` (positional renderer can't emit `if`). Economy-of-mechanism, not expressiveness, keeps init separate.
  - **Init-domination dep-model decision PARKED (user-instructed "revisit"; `tvm_knowledge.md`)**: TVM block-granular+region_cover vs shipped loop-granular carry-edges — BOTH give the guarantee, NOT a correctness fork. Whether TVM rejects `compute_at(init,K)` is `[INFERRED]` not proven; **probe BLOCKED — TVM not built on the desktop**. RFactor doesn't depend on this. `tvm_knowledge.md` tags `[SRC]`/`[PROBE]`/`[INFERRED]` + lists retracted claims.
- **SoftwarePipeline SHIPPED (`dev_1`; Tier B = stage assignment)** — faithful port of TVM `InjectSoftwarePipeline`. Knob = per-child-block `(stages, order)` annotation on parent `BlockNode.annotations`; **version count DERIVED `use_stage−def_stage+1`** → `Buffer.versions`. **Stageable units = ALL direct loop children** (matmul nested ForNode → version derivation reads `ir.dependency.info(leaf)`). Legality = TVM's 2 `ValidatePipelineBody` rules, NO capacity gate. **MEASURED: matmul 59.6%→83.4% MFU.** *(2026-06-06 15:30 ET)*

  - **Fused single-accumulator (90.92% shape) NOT reachable by composing shipped transforms** (3 HW scouts: plain `ComputeAt(wb→ko)`, redaxis-disable, nest-via-ComputeAt — ALL blocked by genuine RAW/carry edges, NOT over-rejection). **Parked "narrow `_check_no_reduction_axis_covered`" premise DISPROVEN**: that check is REDUNDANT (disabling adds 0 accepts; the loop-granular carry-edge independently rejects every NaN case). - **Memset = sibling block, NOT nested `matmul.init`** (`BlockNode.init` dropped; permanently post-`decompose_reduction`). - **Loop-role placement (OP-LOCAL)**: `ForNode` role-neutral; role on `BlockNode.iter_var.role` (=`AXIS_ROLES[axis]`, default PARALLEL); same dim differs across ops. Split/Fuse role-blind; Reorder/compute_at consult. **3 roles**: PARALLEL, ACCUMULATION (reassociable — RFactor needs), SEQUENTIAL (ordered, NOT reassociable). **Reorder** = payload-swap adjacent parent-child ForNode pair; legal iff perfect-nest + no descendant SEQUENTIAL leaf on swap_dim. *(2026-06-07 20:00 ET)*
- **MFU SOTA vs nkipy baselines** (`@baremetal_jit`→HLO→`neuronx-cc -O2`): matmul_lhsT_rhs 90.92% (>86.65%); matmul_lhs_rhs 84%; rmsnorm+matmul ~79-80%; causal attn 13.20%. - **ISA**: `dma_transpose` HBM→SBUF AND SBUF→SBUF, matching dtypes (`dma_copy` tolerates mismatch). `nc_transpose(dst=psum)` IS `nc_matmul(moving=identity, is_transpose=True)`, `data=` NOT `src=`. ## Verification
- Multi-D PSUM trips sim reshape. HW rejects 4D affine-select dst APs (`NCC_INLA001`); `NCC_ISCH714` also fires on aliased `address=` RAW inorder. *(2026-04-23 10:00 ET)*
- **Byte-exact transform verification**: gate = `render(apply(before_IR)) == <hand kernel>`, NOT "roughly the same" (user: "same means same"). Oracle (`_ladder_compare.py`) **AST-canonical** (hoists decls, positional→keyword, sorts kwargs, distributes affine). Topology/sim-only tests hide region bugs. **The hand fixture MUST encode the SPEC shape, NOT the transform's own captured render** — RFactor shipped emitting flat-sibling memset/drain (spec wanted nested per-`ko`); byte-exact + sim BOTH passed because the fixture was captured FROM the deviating render → "byte-exact ✅" meant "matches what it emitted", never "matches spec". Author the fixture from the spec by hand. *(2026-06-09 14:00 ET)*
- **Numeric sim is a WEAKER oracle than dependency-order** — legality = dep-order preservation, NOT numeric luck. memset-after-matmul / drain-sunk-into-K sim "correct" by last-write-wins yet violate producer/consumer order + break HW; a rejecting check is RIGHT even when sim passes. **Behavior-preserving legality refactor → GOLDEN-VERDICT gate, not the suite**: a pure-topological swap silently over-permitted (`illegal→ok`) while suite + sim kernels stayed GREEN — capture each candidate's old verdict first. *(2026-06-04 16:00 ET)*
- **FP32 validation contract**: CPU-sim runs fp32 (inputs cast, SBUF/PSUM→`nl.float32`, fp32 golden); HW keeps original. End-to-end `atol=rtol=5e-3` (`assert_allclose` raises+halts → N PASS+exit 0 = all passed); unit — elementwise `1e-6`, matmul `1e-5`, reduce `1e-4`. *(2026-05-06 17:00 ET)*
