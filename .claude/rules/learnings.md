# Learnings

## Code Quality
- Don't rebuild derived maps in hot paths — cache on `id(obj)` at module level (per-call `inspect.getsource` = 50×+ slowdown). *(2026-04-23 10:00 ET)*
- **No `load_block`/`matmul_block`/`allocate_buffers` gadgets** — tile loops IR-positioned; bare `nl.ndarray`+`nisa.dma_copy`/`nc_matmul` at op fire depth; transpose stays its own op. **`nl.ndarray` N-D explicit** `(P, num_p_tiles, F_contig)`; NO trivial-dim folding; bare `128` (no `nl.par_dim`); `nisa.dma_copy`>`nl.load`. *(2026-04-26 11:00 ET)*
- **Loud failures only**: no silent raises, no `try/except` to adapt around malformed IR — reject bad input. A transform that no-ops on bad input yet returns "success" is the same violation. **Single return per function** (user rejects branching returns in codegen). *(2026-06-01 22:00 ET)*
- **Transform legality = behavior/dep-order + ISA well-formedness ONLY; NEVER resource capacity** (user-locked): a dep-legal kernel that over-subscribes PSUM/SBUF is a VALID transform output — compile/HW profiling prunes it. DO reject malformed IR / dep violations / ISA-impossible ops (Split `MIN_TILE_SIZE`: sub-128 `nc_matmul` malformed). DO NOT gate "won't fit in PSUM" — compile/HW catches it. *(2026-06-06 06:00 ET)*

## Architecture
- **`nkigym/__init__.py` lazy `__getattr__`**: eager re-exports drag heavy deps (`claude_agent_sdk`) into gym workers — keep `__init__` empty of compile/synthesis loads. **Editable pip install pins source path** across shared-venv worktrees: subprocess tests (`sys.executable`) pick up installed path not cwd → `PYTHONPATH=<worktree>/nkigym/src`. *(2026-05-07 13:00 ET)*
- **Neuron allocator** Chaitin-Briggs+zone-aware: `address=` annotations skip `AddressRotation::run` → ~48pp MFU loss. *(2026-04-26 04:00 ET)*
- **Material-blocking**: reading PSUM w/ open material-blocking → `NCC_ISCH714`. Memset PSUM only when `PSUM_DTYPE=fp32`; transpose bf16 PSUM must NOT memset → `NCC_IBIR441`. HW mandates fp32 for `tensor_scalar.operand0/1`, `activation.scale`. *(2026-04-23 10:00 ET)*
- **Path B (own IR) > Path A (fork neuronx-cc)**: Path A ceiling = compiler IR (no online fusion/rotation/SW-pipeline/PSUM-bank addrs). **Online fusion is NO compiler's transform** (TVM `CommReducer` fixed-associative) — *ours* regardless. **Decision: faithfully PORT TVM's schedule transforms into the IR.** Scope: nkigym ≈ pure-Python port of **exactly TVM's TensorIR schedule layer + `arith`** (NOT Relax/TE/topi) — single target, transforms = `tir.Schedule` subset; win is online fusion + agent-legibility, not multi-target. *(2026-06-03 02:00 ET)*
- **arith substrate SHIPPED (Spec 1/4 of TVM port; `nkigym/ir/arith/`)**: faithful TVM `src/arith/` port. **Oracle-verified vs a BUILT TVM**; TVM dep **test-only** (`importorskip`), shipped `nkigym/` imports zero tvm. `can_prove_equal` can't fold `(f//c)*c+f%c→f` — use `detect_iter_map`. Split/Reorder/Fuse arith-thin (delegate to `normalize_block`) → oracle GUARDS not rewrites. *(2026-06-03 12:00 ET)*
  - **Test-slimming = dedup/parametrize, PRESERVE EVERY ASSERTION** (user-locked): `@parametrize` near-identical tests, hoist helpers to `_fixtures.py`, merge corpus constants; arith `_logic`/oracle pairs keep per-function `importorskip("tvm")`.
- **Repo scope**: `NKIOps` + `KernelIR` (`ir/ir.py`, wraps `DimensionAnalysis`+`KernelTree`) build + `render` to NKI; `.dump(cache_dir)` emits markdown + tree PNG + `kernel.py`. *(2026-06-01 22:00 ET)*
- **NO trip-1 ForNodes anywhere**: a dim's iteration = factorization `(trip₀…trip_{k-1}, tensorize_size)`, product = extent; ForNode exists ONLY when trip > 1; innermost `tensorize_size` = ISA-leaf width. **Three redundant-but-load-bearing layers** per iteration: `iter_var.dom` (full extent), ForNode trips, `BufferRegion (lo,width)`; invariant `Π(trip)×width == dom`; redundancy serves compute_at region-regen. *(2026-06-01 02:00 ET)*
- **BlockNode IR** (`tree.py`, nx.DiGraph): payload union `BlockNode|ForNode|ISANode`, own `.label()`. `BlockNode` (TVM `SBlock`) owns `iter_vars` (axis+`dom`+role), `iter_values` (affine loop_vars→iter_var), `reads`/`writes`, `alloc_buffers`. `axis_map` concrete→abstract: tensorize/Split translate `d2`→op-axis `N` for `OPERAND_AXES`/`MIN_TILE_SIZE`. *(2026-05-29 09:00 ET)*
- **SSA ops backend**: `f_nkigym` is SSA — `out = NKIOp()(src=…)`, allocate-and-return; `NKIAlloc`/`dst=`/authored memset GONE. Memset synthesized in `canonical_build` from `RMW_OPERANDS`. `NKITensorCopy` explicit (multi-output breaks SSA). **Dtype** (`Buffer.physical_dtype`): PSUM accum alloc ALWAYS fp32 (ONLY hardcode); every other buffer = FIRST HBM input's dtype. *(2026-06-01 22:00 ET)*
- **LCA buffer placement, re-derived after EVERY transform** (`place_buffers`): each `Buffer` on the LCA BlockNode of all blocks touching it — the PSUM-hoist lever (compute_at co-locating memset/matmul/tensor_copy under shared M,N drops LCA). `shared_hbm` pinned at root (else re-alloc → NaN). **Location enum `{shared_hbm, sbuf, psum}`** (literal IS var prefix); renderer keys operand dimensionality off it (HBM 2D, SBUF/PSUM 3D). *(2026-06-02 10:00 ET)*
- **Producer-consumer deps = sidecar DAG** (`ir/dependency.py`, networkx; ISA-leaf/ForNode nids). Tree stays block-structural — don't reparent loops to encode RAW. **ONE notion: same buffer + overlapping region + RAW/WAW/WAR**; legality = leaf positions + spans. *(2026-06-03 02:00 ET)*
  - **Ill-defined-edge law**: `a→b` meaningless when one nests in the other → NO node↔loop edge. Resolution = **symbolic unroll** (pure instruction→instruction); loop-carry via span-promotion. **Carry axis role-based**: `producer→outermost-loop-of-axis` edge only when enclosing loop non-PARALLEL AND axis ∉ output chunk.
  - **Direction bug — FIXED** (`df43f44`): NEVER rebuild `Dependency` on the MOVED tree — tautological (RAW→forward WAR → no backward edge → NaN). Freeze directions from ORIGINAL `ir.dependency`, eval positions on moved tree via pure topological `first_backward_edge_for_insertion`. arith = WRONG layer.
- **Codegen direction: ir → codegen, never reverse.** Renderer takes IR primitives (tree/block nodes, `BufferRegion`s), not `KernelIR` — one import direction kills circular deps. **loop_var naming = model B**: dense `i_d{dim}_{N}`, N = ordinal among dim's ForNodes outer→inner. **Sibling order = dataflow**: splice `add_edge` at original child index (else load renders AFTER matmul). *(2026-06-01 04:00 ET)*
- **Transforms shipped: Split + Fuse + Reorder + ComputeAt/ReverseComputeAt** (`nkigym/transforms/`). `Transform` base: `analyze(ir)→list[TransformOption]`, `apply(ir,option)→KernelIR` (deep-copy+mutate, re-checks legality → loud `TransformLegalityError`). **Split respects `MIN_TILE_SIZE`** (matmul K/M/N all 128) → M/K@128 unsplittable, N(512)→128. **compute_at**: shared `_move(...,is_reverse)`, faces differ ONLY in dep-check direction; **TVM region-regen NOT trip-collapse** (`SolveBlockVarDomain` inverts access↔required region, `MakeNewLoop` regenerates residual loops). Invariants: gather enclosing-ancestor loops (else offset→0→NaN); `compact_shapes` anchor = loops enclosing all touchers. *(2026-06-02 10:00 ET)*
- **SoftwarePipeline SHIPPED (`dev_1`; Tier B = stage assignment)** — faithful port of TVM `InjectSoftwarePipeline`. Knob = per-child-block `(stages, order)` annotation on the parent `BlockNode.annotations`; **version count DERIVED `use_stage−def_stage+1`** → on `Buffer.versions` (`physical_shape()` single-source, tile dim ×versions). **Stageable units = ALL direct loop children (BlockNode OR ForNode-nest)** — matmul is a nested ForNode, so version derivation reads `ir.dependency.info(leaf).reads/writes` (nest has NO BlockNode carrying regions). Legality = TVM's 2 `ValidatePipelineBody` rules, NO capacity gate. **MEASURED: matmul 59.6%→83.4% MFU** via `[4 atoms, SoftwarePipeline(11,(0,0,1))]`, byte-exact to validated `rotate_only`. **Increment-2 SKEW renderer DEFERRED** (+0.8pp; compiler auto-pipelines): needs precedence-aware Expr formatter. *(2026-06-06 15:30 ET)*
  - **TRACE search EXHAUSTED — 83.5% is the shipped-transform ceiling** (HW-measured, ~12 candidates): loop-reorder M>K>N weight-reuse = NO gain (compiler already reuses stationary weight, 83.06 vs 83.00); pipeline `stages=(0,1,1)` slightly WORSE (83.26); depth-3 accumulator OOMs (3 MiB > 2 MiB, HW prunes — validates no-capacity-gate); **N-outermost (N>M>K) blocked — the canonical FULL-TENSOR drain won't sink past the N loop** (ComputeAt produces ONE drain, not the per-(n,m) tiled drain the SOTA needs; only `target_loop_nid=store-loop` legal). Path to 90.92% SOTA = rfactor (2-stage K→SBUF combine) + M-blocked/N-tiled drain, NEITHER shipped. Don't re-explore reorder/depth/N-block on matmul — measured dead ends.
- **Memset = sibling block, NOT nested `matmul.init`** (`BlockNode.init` dropped; folding gave duplicate loops + INVERTED edge). init/compute/drain at THREE loop levels = TVM's *post-`decompose_reduction`* state. A free memset CAN sink into K → needs `memset→K-loop` RAW edge. **Reduction-dim-not-in-output**: NKIOps declare `REDUCE_COMBINATOR`; if `blocking_dims-output_dims` non-empty + trip>1 → memset + `tensor_tensor`. *(2026-06-03 02:00 ET)*
- **Op axis names follow `OPERAND_AXES`, not matmul names**: `NKILoad.OPERAND_AXES={"P","F"}`, so a load's iter_var axes are `{"P","F"}` even when concrete dim is `d1`. *(2026-05-17 14:00 ET)*
- **Loop-role placement (OP-LOCAL)**: `ForNode` role-neutral; role on `BlockNode.iter_var.role` (=`AXIS_ROLES[axis]`, default PARALLEL); same dim differs across ops. Split/Fuse role-blind; Reorder/compute_at consult. **Reorder** = payload-swap adjacent parent-child ForNode pair (self-inverse); legal iff perfect-nest + no descendant SEQUENTIAL leaf on swap_dim. *(2026-05-29 09:00 ET)*
- **MFU SOTA vs nkipy baselines** (`@baremetal_jit`→HLO→`neuronx-cc -O2`): matmul_lhsT_rhs 90.92% (>86.65%); matmul_lhs_rhs 84%; rmsnorm+matmul ~79-80%; causal attn 13.20%. Attention (trn2): compiler flash/vanilla neither >~18%; hand `nkilib.attention_cte` D=128 bf16 30/46% @ 2k/16k. **MAC = 4·SEQ²·D**; `compute_mac_count` = `K·M·N` per matmul (1 MAC = 2 FLOPS). **Profiler `throttle_avg_util_limit` NOT HW ceiling** (activity-weighted); real gap = `tensor_engine_active_time_percent`. Roofline eff = `100*MFU/mfu_max_achievable`; low-MFU+high-ceiling splits by MBU: high MBU = mem-bound, **low MBU = serialization stall**. *(2026-06-05 12:00 ET)*
- **ISA**: `dma_transpose` HBM→SBUF AND SBUF→SBUF, matching dtypes (`dma_copy` tolerates mismatch). `nc_transpose(dst=psum)` IS `nc_matmul(moving=identity, is_transpose=True)`, kwarg `data=` NOT `src=`. `nl.divide`→`reciprocal+multiply`. ## Verification
- Multi-D PSUM trips sim reshape. HW rejects 4D affine-select dst APs (`NCC_INLA001`); `NCC_ISCH714` also fires on aliased `address=` RAW inorder. `nisa.dma_copy` has NO transpose — use `nisa.dma_transpose`. *(2026-04-23 10:00 ET)*
- **Byte-exact transform verification**: gate = `render(apply(before_IR)) == <hand kernel>`, NOT "roughly the same" (user: "same means same"). Oracle (`_ladder_compare.py`) **AST-canonical** (hoists decls, positional→keyword, sorts kwargs, distributes affine). Topology/sim-only tests hide region bugs — render+sim caught all 5 move-engine bugs. *(2026-06-02 10:00 ET)*
- **Numeric sim is a WEAKER oracle than dependency-order** — legality = dep-order preservation, NOT numeric luck. memset-after-matmul / drain-sunk-into-K sim "correct" by last-write-wins yet violate producer/consumer order + break HW; a rejecting check is RIGHT even when sim passes. **Behavior-preserving legality refactor → GOLDEN-VERDICT gate, not the suite**: a pure-topological swap silently over-permitted (`illegal→ok`) while suite + 16 sim kernels stayed GREEN — capture each candidate's old verdict first, require exact match. *(2026-06-04 16:00 ET)*
- **FP32 validation contract**: CPU-sim runs fp32 (inputs cast, SBUF/PSUM→`nl.float32`, fp32 golden); HW keeps original. End-to-end `atol=rtol=5e-3` (`assert_allclose` raises+halts → N PASS+exit 0 = all passed); unit — elementwise `1e-6`, matmul `1e-5`, reduce `1e-4`. *(2026-05-06 17:00 ET)*
